Detailed Description

one monadic scalar job

A PJob is a computation that can be carried out in parallel on different cores. The PJob_XXX structs defined in this file contain the essential information needed to do the job.

A Job is typically implemented by iterating some Cell function along the ravels of the result Z, the right argument B, and possibly (for dyadic Cell functions) the left argument A.
