given 4 points A=(Ax, Ay), B=(Bx, By), C=(Cx, Cy), and D=(Dx, Dy) in the plane, return the factor beta so that the intersection point M=(Mx, My) of the lines A-C and B-D is given by

M = (Mx, My) == A + alpha*(D-A) == B + beta*(C - B). Note that:

A \ B \ / M / \ / \ C D Ax + alpha*(Dx-Ax) == Bx + beta*(Cx-Bx) Ay + alpha*(Dy-Ay) == By + beta*(Cy-By)
