◆ use()

filebuf * ErrOut::use ( )
inline

a helper function telling whether the constructor for CERR was called if CERR is used before its constructor was called (which can happen in when constructors of static objects are called and use CERR) then a segmentation fault would occur.

We avoid that by using get_CERR() instead of CERR in such constructors. get_CERR() checks used and returns cerr instead of CERR if it is false.
