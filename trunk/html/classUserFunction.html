std::vector<Function_PC> UserFunction::line_starts
protected

Offsets to the first token in every line (for jumps). lines[0] points to the last line, which is automatically added and is TOK_RETURN_VOID for void functions and TOK_RETURN_VALUET for functions returning a value.

An N line function:

[0] R←A FOO B [1] L1: xxx ... [N-1] yyy

will, for example, have a N+1 element jump vector:

[0] goto end of function -----------—+ [1] L1: xxx | ... | [N-1] yyy | [N] TOK_RETURN_SYMBOL or TOK_RETURN_VOID <–+
